This project is a **Restaurant Management System** developed using **Flask** and **MongoDB** to manage restaurant operations such as menu handling, table reservations, online ordering, gallery management, and admin control.

>  Note: Due to platform submission limitations, only documentation files are uploaded. The complete source code is available upon request.

---

## Project Objective
To design and develop a full-stack web application that simplifies restaurant operations and provides separate access for customers and administrators.

---

##  Features

###  Customer Module
- User registration and login
- Browse menu by category
- Online table reservation system
- View and cancel reservations
- Contact restaurant
- Gallery viewing

###  Admin Module
- Secure admin authentication
- Dashboard with statistics
- Menu category and item management (CRUD)
- Reservation approval and management
- Order management
- Gallery image upload and deletion
- Customer message management

---

## Tech Stack
- **Backend:** Flask (Python)
- **Database:** MongoDB
- **Frontend:** HTML, CSS, Jinja2
- **Authentication:** Session-based authentication
- **Security:** Password hashing using Werkzeug
- **Image Handling:** Pillow for resizing and storage
