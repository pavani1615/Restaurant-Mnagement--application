# Restaurant Management System

A full-stack web application built using **Flask and MongoDB** that manages restaurant operations such as menu management, reservations, online ordering, gallery, and admin control.

## Features

###  User Features
- User registration and login
- View restaurant menu by category
- Online table reservation system
- View and manage personal reservations
- Contact restaurant via contact form
- Browse gallery and restaurant details

###  Admin Features
- Secure admin dashboard
- Add, edit, delete menu categories and items
- Manage reservations (approve / cancel)
- View and manage customer orders
- Upload and manage gallery images
- View customer contact messages

##  Tech Stack
- **Backend:** Flask (Python)
- **Database:** MongoDB
- **Frontend:** HTML, CSS, Jinja2
- **Authentication:** Session-based authentication
- **Security:** Password hashing with Werkzeug
- **Image Handling:** File upload and resizing
